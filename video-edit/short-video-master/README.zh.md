## 核心原则

短视频剪辑不是简单拼素材，而是设计留存。每一秒都要回答一个问题：观众为什么要继续看？

## 基础格式

- 默认比例：9:16。
- 开头 1 到 3 秒必须有钩子。
- 字幕要短、清楚、高对比。
- 音频先保证人声清楚，再考虑音乐氛围。
- 每个剪切点都要改变信息、情绪或节奏。

## 常用结构

1. 问题 -> 紧张 -> 尝试 -> 揭晓 -> 反应。
2. 变美前 -> 过程压缩 -> 关键转折 -> 变美后。
3. 错误示范 -> 正确做法 -> 证据 -> 总结。
4. 先给结果 -> 倒回过程 -> 最终爽点。
5. 冲突台词 -> 升级 -> 反转 -> 悬念。

## 竖屏短剧剪辑规则

- 不要长铺垫，直接进冲突。
- 第一句台词要制造矛盾、身份变化或疑问。
- 情绪用近景，证据用特写。
- 反应镜头是标点，不是填空。
- 每场戏结尾要有反转、选择、秘密或压力。

## 变美类视频规则

- 必须看清楚“前、过程、后”。
- 不要羞辱变美前的状态。
- 皮肤不要磨到失真。
- 变美后揭晓要卡在音乐重拍、转身、擦镜、开门、眨眼、镜子出现等动作上。
- 揭晓后给 2 到 3 个证明角度。

## 字幕规则

- 一行只表达一个意思。
- 能用短词就不用长句。
- 重点词可以高亮，但不要全屏都是字。
- 字幕不能挡住脸、产品、手部动作和关键证据。
- 对话类视频要去掉无意义口癖。

## 输出模板

AI 使用本技能时，应输出：

- 视频结构；
- 秒级节奏表；
- 镜头建议；
- 字幕建议；
- 音乐和音效建议；
- 完播风险；
- 导出规格。
