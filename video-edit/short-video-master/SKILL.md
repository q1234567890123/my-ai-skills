## Core Rule

Short video editing is retention design. Every second must give the viewer a reason to continue watching.

## Default Format

- Aspect ratio: 9:16.
- First 1-3 seconds: show result, conflict, visual surprise, or curiosity gap.
- Captions: short, high contrast, mobile readable.
- Audio: dialogue clarity first, music rhythm second.
- Cuts: each cut should change information, emotion, or rhythm.

## Common Structures

- Problem -> tension -> attempt -> reveal -> reaction.
- Before -> process compression -> turning point -> after.
- Mistake -> correction -> proof -> takeaway.
- Hook result -> rewind process -> final payoff.
- Conflict line -> escalation -> reversal -> cliffhanger.

## Beauty Video Rules

- Make the before, process, and after visually clear.
- Do not humiliate the before state.
- Preserve natural skin texture.
- Put the reveal on a beat, turn, wipe, blink, mirror reveal, or door open.
- Show proof angles after the reveal.

## Vertical Drama Rules

- Start inside the scene, not with long setup.
- First line should create conflict, status shift, or unanswered question.
- Use close-ups for emotion and inserts for evidence.
- End with decision, secret, reversal, or unresolved pressure.

## Review Checklist

- Is the first frame interesting?
- Can the video be understood muted?
- Is the payoff visible before attention drops?
- Are captions readable on a small phone?
- Does the ending create replay, comment, save, follow, or next-episode desire?
