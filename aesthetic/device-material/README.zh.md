# Device Material 中文版

路径：`aesthetic/device-material/`

## 这个技能是干什么的

这个技能用于解释医美材料和设备的基础原理，例如玻尿酸、胶原刺激类材料、肉毒素概念、水光、光电设备、激光、射频、超声、IPL 等。

它适合做科普和咨询解释，不用于临床参数和治疗决策。

## 安全边界

本技能不提供：

- 医学诊断；
- 设备能量参数；
- 注射剂量；
- 注射层次；
- 禁忌症排除结论；
- 并发症处理；
- 具体品牌疗效承诺。

所有治疗选择必须由正规机构和有资质医生面诊判断。

## 材料分类

- 填充/支撑类：主要用于容量补充、轮廓塑形、支撑改善。
- 补水/肤质类：主要用于水润度、细纹、光泽、肤质改善。
- 肌肉活动调节类：通过影响特定肌肉活动来改善动态纹或形态。
- 胶原刺激类：偏渐进式，强调组织质量和胶原相关改善。
- 线材类：偏机械支撑和组织反应，强依赖适应证和操作。

## 设备分类

- 激光/光类：常用于色素、泛红、脱毛、嫩肤、重建表面质感。
- IPL：广谱强脉冲光，适合部分肤色、泛红、色沉、脱毛场景。
- 射频：热作用相关，常用于紧致、质地、胶原重塑概念。
- 超声：聚焦能量，用于更深层紧致或提升概念。
- 微针：通过微损伤刺激修复，通常需要专业操作。
- 其他身体设备：可能涉及塑形、循环、肌肉刺激等。

## 解释框架

解释任何材料或设备时，按这个顺序：

1. 它解决什么问题。
2. 它作用在哪类组织或信号。
3. 结果是即时、延迟还是渐进。
4. 自然度如何。
5. 是否可逆、恢复期多长、维护周期如何。
6. 操作者经验影响有多大。
7. 哪些必须面诊后决定。

## 风险沟通

用普通人能懂的话说风险：

- 常见短期反应：红、肿、痛、淤青、干燥。
- 操作相关风险：不对称、凹凸、僵硬、过度。
- 设备相关风险：烫伤、色沉、瘢痕、眼部风险。
- 材料相关风险：结节、炎症、感染、血管事件。
- 个人因素：肤色、用药、孕期、免疫情况、近期治疗史。

不要制造恐慌，但也不能说“完全没风险”。

## 对比模板

对比两个项目时，输出：

- 适合解决的问题；
- 不适合的人群或情况；
- 起效速度；
- 自然度；
- 维护周期；
- 恢复期；
- 可逆性；
- 需要问医生的问题。
