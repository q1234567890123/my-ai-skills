# Medical Design 中文版

路径：`aesthetic/medical-design/`

## 这个技能是干什么的

这个技能用于医美美学设计、面部比例分析、审美方向表达、抗衰优先级、术前术后对比评价、客户沟通话术。

它不做医学诊断，不给剂量，不讲注射层次，不给设备参数，不替代医生面诊。

## 安全边界

本技能只做审美分析和表达建议，不提供：

- 医学诊断；
- 是否适合治疗的结论；
- 注射剂量；
- 注射层次；
- 设备能量参数；
- 药物建议；
- 手术步骤；
- 并发症处理方案。

所有临床决策必须由有资质医生面诊后完成。

## 面部分析框架

从大到小看：

1. 整体印象：年龄感、疲态、亲和力、锐度、柔和度。
2. 比例：上中下庭、脸宽、下巴、额头、面中关系。
3. 轮廓：额头、太阳穴、颧部、下颌、下巴、颈部过渡。
4. 五官协调：眼、鼻、唇、眉、笑容。
5. 动态表情：笑、说话、皱眉、眼神变化。
6. 皮肤质量：色泽、纹理、毛孔、泛红、色沉、松弛、水润度。

## 设计优先级

1. 健康感和皮肤质量。
2. 减少疲态。
3. 比例和轮廓平衡。
4. 五官细节优化。
5. 镜头表现力。
6. 流行趋势最后考虑。

不要因为客户只盯着一个部位，就只分析那个部位。很多“鼻子不好看”“嘴不好看”，实际可能来自面中、下巴、轮廓、表情或发型影响。

## 沟通规则

建议使用：

- “柔和一点”；
- “更平衡”；
- “改善疲态”；
- “增强支撑感”；
- “保留辨识度”；
- “分阶段更稳妥”。

避免使用：

- “你这里很丑”；
- “必须做”；
- “不做会垮”；
- “保证自然”；
- “完全没风险”。

## 方案模板

AI 输出医美设计建议时，应包含：

- 审美目标；
- 当前影响整体美感的主要信号；
- 优先级排序；
- 保守路线；
- 进阶路线；
- 发型、妆容、光线、体态等非医美支持；
- 复查和阶段评估建议。

## 术前术后评价

评价时看：

- 是否还像本人；
- 是否改善整体协调，而不是只改善某个角度；
- 表情是否自然；
- 是否过度填充或僵硬；
- 光线、角度、妆容、镜头是否可比。
