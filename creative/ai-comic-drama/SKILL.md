## Core Rule

AI comic drama must protect visual continuity. Script, character bible, camera language, emotion beats, and prompt vocabulary must all prevent identity drift.

## Required Package

When creating a project, output:

1. Logline.
2. Character bible.
3. Episode outline.
4. Scene or panel table.
5. Prompt pack.
6. Continuity checklist.
7. Ending hook.

## Story Rules

- Start with conflict or an imageable question.
- Give every scene one emotional turn.
- Use concrete props as plot anchors.
- Avoid exposition that cannot be shown visually.
- Keep dialogue short.
- End with decision, secret, reversal, or emotional pressure.

## Prompt Rules

- Put identity anchors early.
- Repeat stable wardrobe and accessory details.
- Keep art style, lens, palette, and lighting consistent.
- Specify shot size and camera angle.
- Use visible behavior to describe emotion.
- Use negative prompts for identity drift, wrong outfit, extra limbs, distorted faces, and unreadable text.
